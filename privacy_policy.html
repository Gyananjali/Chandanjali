Privacy Policy for Chandanjali

Effective Date: 22 Feb 2026 App Name: Chandanjali Developer / Legal Entity: Gyananjali

Chandanjali is an offline Sanskrit Chandas analysis application developed for educational purposes.

1. Information Collection

Chandanjali does not collect, store, transmit, or share any personal information.

  • No user registration is required.
  • No personal data is collected.
  • No analytics or tracking tools are used.
  • No advertising SDKs are integrated.

2. User Input

All Sanskrit text entered into the application is processed locally on the user's device. No input data is transmitted to external servers.

3. Internet Usage

The application functions completely offline and does not require an internet connection.

4. Data Sharing

Since no data is collected, no information is shared with third parties.

5. Children's Privacy

This application does not knowingly collect personal information from children or any users.

6. Security

As no personal data is collected or stored, there is no external data storage associated with this application.

7. Changes to This Policy

Any updates to this Privacy Policy will be reflected on this page with an updated effective date.
